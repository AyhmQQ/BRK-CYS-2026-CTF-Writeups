# BRK-CYS 2026 CTF - Official Writeups

This repository contains official writeups for challenges authored in the **BRK-CYS 2026 CTF**.

These challenges are designed to go beyond standard CTF difficulty, focusing on **advanced cryptographic concepts**, **mathematical modeling**, and **real-world attack strategies**.
